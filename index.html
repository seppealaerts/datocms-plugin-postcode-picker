About

Postcode Picker is a custom DatoCMS field extension plugin that allows content editors to easily select multiple postcodes using an intuitive autocomplete search interface. Powered by the GeoNames API, it provides real-time search results for postcodes and cities worldwide.

Installation

  1. Build the plugin:
    npm install
    npm run build
  2. Install in your DatoCMS project:
    • Go to Settings ‚Üí Plugins
    • Click "Add a plugin" ‚Üí "Install from URL"
    • Or upload the built plugin

Usage

  1. Configure the plugin:
    • Go to Settings ‚Üí Plugins ‚Üí Postcode Picker
    • Enter your GeoNames username (optional, for higher rate limits)
    • Set the country code (ISO 3166-1 alpha-2, e.g., BE, NL, FR)
    • Toggle "Group cities with same postcode" if desired
    • Click "Save"
  2. Add the field extension:
    • Add a new JSON field to your model
    • In field settings, scroll to "Field extensions"
    • Click "Add field extension" and select "Postcode Picker"
  3. Use the picker:
    • Type to search for postcodes or city names
    • Select multiple postcodes from the dropdown
    • Selected postcodes are displayed as chips

Data Format

Selected postcodes are stored as a JSON string:

[
{ "postcode": "1000", "city": "Brussels" },
{ "postcode": "2000", "city": "Antwerp" }
]

When city grouping is enabled, multiple cities are stored comma-separated in the city field.

GeoNames API

The plugin uses the GeoNames API (free tier). For higher rate limits, you can configure your GeoNames username in the plugin settings.

Get a free GeoNames account: https://www.geonames.org/login
